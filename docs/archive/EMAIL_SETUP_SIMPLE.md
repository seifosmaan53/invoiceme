# 📧 Email Setup - Simple Guide

## ⚡ Quick Answer

**Setup time:** 30 seconds  
**On new device:** 30 seconds  
**Frequency:** Once per device

---

## 🚀 Setup (30 Seconds)

```bash
cd backend
npm run setup:email
```

**Done!** Emails are configured.

---

## 🔄 On New Device?

### Easiest: Run Script Again
```bash
cd backend
npm run setup:email
```

### Faster: Use Saved Credentials
1. Save credentials from first setup
2. On new device: Paste into `.env` file
3. Takes 1 minute

---

## 💡 Pro Tip: Save Credentials

**After first setup:**
- Copy the credentials shown
- Save in password manager
- Reuse on all devices!

**This way:** Set up once, reuse forever! 🎉

---

## ✅ Test It

```bash
npm run test:email
```
